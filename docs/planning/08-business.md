# 08-business.md: Business Model & GTM Strategy

## MVP 캡슐 (요구사항 핵심 10줄)

> **목표**: 흩어진 추억을 안전하게 한 곳에 보존  
> **페르소나**: NAS+HDD+스마트폰에 20만+ 장 분산된 IT 개발자  
> **핵심 기능**: F1 중복감지(2-Stage), F2 베스탈 선별(AI추천+사용자확인), F4 내보내기(단순복사)  
> **North Star**: 중복 그룹 발견율 95%+ / 오탐율 5% 미만  
> **제약**: 20만 장 전체 스캔 30분 이내, 폴더별/기간별 선택 스캔, 증분 스캔  
> **위험**: 보정본 감지 정확도 (pHash 임계값)  
> **완화 전략**: Hamming Distance 임계값 A/B 테스트 (8 vs 10 vs 12)  
> **범위 외**: 메타데이터 통합(v2), 동영상(v2), 모바일(v2)  
> **Input Metric**: 스캔 완료율, 사용자 판정 완료율  
> **다음 단계**: `/screen-spec`으로 화면 명세 생성

## 1. 비즈니스 모델: Freemium

### 1.1 모델 개요

```
┌──────────────────────────────────────────────────┐
│           Freemium Monetization                   │
├──────────────────────────────────────────────────┤
│                                                  │
│  Free Tier (90%)          Premium Tier (10%)    │
│  ├─ 기본 감지             ├─ 무제한 스캔       │
│  └─ 수동 선택             ├─ 자동 베스탈       │
│                           ├─ 일괄 내보내기     │
│  Conversion Strategy:      └─ 우선 지원        │
│  1. 작은 제한으로 가치 증명
│  2. 프리미엄 기능 체험
│  3. 자연스러운 전환
│
│  Revenue Model:
│  - Subscription: $9.99/월 또는 $79.99/년
│  - Team Plan (v2): $19.99/월
│  - Enterprise (v3): 연간 계약
│
└──────────────────────────────────────────────────┘
```

### 1.2 Freemium 경계선 설정

#### Free Tier 제한

| 기능 | Free | Pro | 사유 |
|------|------|-----|------|
| **스캔 가능 용량** | 1,000장 | 무제한 | MVP 테스트, 부하 제어 |
| **병렬 스레드** | 2개 | 8개 | 서버 리소스 절약 |
| **AI 베스탈 추천** | ✅ | ✅ | 가치 증명 (전체) |
| **수동 베스탈 선택** | ✅ | ✅ | 필수 기능 (전체) |
| **내보내기 (Copy)** | ✅ | ✅ | 핵심 기능 (전체) |
| **Move (원본 삭제)** | ❌ | ✅ | 위험도 높음 (결정 후) |
| **일괄 내보내기** | ❌ | ✅ | 편의 기능 |
| **메타데이터 통합** | ❌ | ✅ | v1.1+ 기능 |
| **Soft Delete (휴지통)** | ✅ (30일) | ✅ | 안전성 (전체) |
| **우선 지원** | ❌ | ✅ | 서비스 (유료) |

**합리성 검증**
- Free 티어: 실제 사용 가능 (1,000장 = 사소한 사용자도 테스트)
- Pro 로직: 대용량 + 자동화 = 전문가 + 기업
- 심리 가격: $9.99 = "저가 구독" 인식 (월 커피 가격)

#### 전환 유도 메커니즘

```
1. 스캔 제한 도달 시 (1,000장)
   → "Pro: 무제한 스캔 시작" 배너 표시
   
2. 내보내기 후
   → "더 빠른 속도 원하세요?" 팝업
   
3. 사용 1주일 후
   → 이메일: "Pro 사용자는 50% 절약"
   
4. 매달 1회
   → 스캔 완료 후: "Pro 14일 무료 체험"
```

## 2. Free Tier 비즈니스 전략

### 2.1 목표

**Free Tier의 역할**
```
수익 아님 ← 사용자 확보 & 검증 → 전환 경로

1. 마케팅 비용 절감 (word-of-mouth)
2. 실제 사용 데이터 수집
3. 제품 개선 반복 (사용자 피드백)
4. Premium 가치 증명
```

### 2.2 Free 사용자 특성

| 수용자 | 비율 | 행동 | 전환 가능성 |
|--------|------|------|-----------|
| **호기심** | 40% | 시도만 하고 이탈 | 낮음 (5%) |
| **가벼운 사용** | 35% | 월 1~2회 사용 | 중간 (15%) |
| **헤비 사용** | 20% | 주 1회 이상 | 높음 (50%+) |
| **불만** | 5% | 피드백 제공 | 낮음 (10%) |

**전략: 35% + 20% 타깃화 (유료 전환 25~40%)**

## 3. Pro Tier 가격 책정

### 3.1 가격 전략

#### 월간 구독 vs 연간 구독

```
월간: $9.99/월
      └─ 심리적 저항 낮음
      └─ 쉬운 취소
      └─ 여름 사용자 확보
      
연간: $79.99/년 (33% 할인)
      └─ 월 $6.67 환산
      └─ 장기 사용자 확보
      └─ 현금흐름 개선
      └─ 초기 구독 인센티브
```

#### 가격 근거 (가치 기반)

```
OptiShot Pro의 가치:
1. 시간 절약: 20만 장 정리 = 30시간 × $50/시간 = $1,500
   → Pro가 1개월 사용해도 ROI 16,000배
   
2. 위험 회피: 중복 삭제 실수 방지 = $0~$1,000 (감정적 손실)
   → Pro = 안전장치 비용
   
3. 하드웨어: 외장HDD 절약 = $50~$100/TB × 50GB = $250+
   → Pro = 비용 절약 도구

결론: $9.99 = 저가
```

### 3.2 할인 전략

| 시나리오 | 할인 | 목적 |
|---------|------|------|
| **초기 구독** | 50% off 1개월 | 첫 전환 유도 |
| **연간 구독** | 33% off (79.99) | 장기 commitment |
| **팀 플랜** (v2) | 50% per user | 기업 유입 |
| **학생** | 50% off | 커뮤니티 구축 |
| **오픈소스 기여자** | 무료 | 커뮤니티 선의 |

## 4. Revenue Projections

├─ Pro Subscriber: 100명 × $9.99 = $999
├─ Annual Upfront: 50명 × $79.99 = $3,999 (1회)
└─ Total MRR: ~$1,000 (안정화 시)

ARR (Annual Recurring Revenue):
├─ Monthly: $999 × 12 = $11,988
├─ Annual: $3,999 × 1 = $3,999

비용:
├─ 개발: $0 (자체 개발)

## 8. 리스크 & 완화 전략

### 8.1 주요 비즈니스 리스크

| 리스크 | 영향 | 확률 | 완화 |
|--------|------|------|------|
| **경쟁** | 높음 | 높음 | NAS 전문화 + 커뮤니티 |
| **이탈** | 중간 | 높음 | 강력한 유지율 (이메일) |
| **신뢰도** | 높음 | 중간 | 오픈소스 + 명확한 문서 |
| **스케일** | 높음 | 낮음 | 클라우드 인프라 (AWS) |
| **규제** | 낮음 | 낮음 | 글로벌 법무 자문 |

### 8.2 리스크 완화 플랜

**경쟁 심화**
```
→ 차별화: NAS 원주민 지원 + 커뮤니티 중심
→ 속도: AI 추천 고도화 (메타데이터 학습)
```

**사용자 이탈**
```
→ 이메일 시퀀스: 스캔 1주일 후, 1개월 후 체크인
→ 기능 요청: 사용자 피드백 기반 월간 업데이트
```

**신뢰도 문제**
```
→ 보안: SOC2 준비 (Y2)
→ 투명성: 월간 개발 로그 공개
→ 데이터: "절대 수집 안 함" 명시 (약관)
```
