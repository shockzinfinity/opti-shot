# 06-screens.md: Screen Definitions

## MVP 캡슐 (요구사항 핵심 10줄)

> **목표**: 흩어진 추억을 안전하게 한 곳에 보존  
> **페르소나**: NAS+HDD+스마트폰에 20만+ 장 분산된 IT 개발자  
> **핵심 기능**: F1 중복감지(2-Stage), F2 베스탈 선별(AI추천+사용자확인), F4 내보내기(단순복사)  
> **North Star**: 중복 그룹 발견율 95%+ / 오탐율 5% 미만  
> **제약**: 20만 장 전체 스캔 30분 이내, 폴더별/기간별 선택 스캔, 증분 스캔  
> **위험**: 보정본 감지 정확도 (pHash 임계값)  
> **완화 전략**: Hamming Distance 임계값 A/B 테스트 (8 vs 10 vs 12)  
> **범위 외**: 메타데이터 통합(v2), 동영상(v2), 모바일(v2)  
> **Input Metric**: 스캔 완료율, 사용자 판정 완료율  
> **다음 단계**: `/screen-spec`으로 화면 명세 생성

## 1. S1: 대시보드 (Dashboard)

### 1.1 목적

앱 시작 시 전체 요약을 보여주고, 다음 액션(스캔/리뷰/설정)으로 진입하는 진입점

### 1.2 주요 컴포넌트

#### 헤더 (상단)

```
┌─────────────────────────────────────────────────────┐
│  🎨 OptiShot Logo          [⚙ Settings] [? Help]   │
└─────────────────────────────────────────────────────┘
```

**요소**
- 로고 + 앱명 (좌측)
- 설정/도움말 아이콘 (우측)

#### 요약 통계 (메인 영역)

```
┌─────────────────────────────────────────┐
│  📊 전체 요약                            │
├─────────────────────────────────────────┤
│                                         │
│  ┌────────────┬────────────┐           │
│  │ 총 사진    │ 중복 그룹  │           │
│  │ 200,000장  │ 5,000개    │           │
│  └────────────┴────────────┘           │
│                                         │
│  ┌────────────┐                        │
│  │절약 가능   │                        │
│  │용량 50GB   │                        │
│  └────────────┘                        │
│                                         │
└─────────────────────────────────────────┘
```

**데이터 소스**
- `Database.FilesTable.Count()` → 총 사진
- `Database.GroupsTable.Count()` → 중복 그룹
- `SUM(Group.DuplicateSize - Group.MasterSize)` → 절약 가능 용량

#### 최근 스캔 (메인 영역 하단)

```
┌─────────────────────────────────────────┐
│  🔍 최근 스캔                            │
├─────────────────────────────────────────┤
│  2024-04-15 14:30 (30분 전)            │
│  ├─ 처리 파일: 200,000장               │
│  ├─ 발견 그룹: 5,000개                 │
│  └─ 예상 절약: 50GB                    │
│                                         │
│  [설정 확인] [결과 보기]                │
└─────────────────────────────────────────┘
```

#### 액션 버튼 (하단)

```
┌─────────────────────────────────────────┐
│  [🚀 스캔 시작] [📂 최근 스캔]          │
│                                         │
│  [🗑 휴지통]   [⚙ 설정]                │
└─────────────────────────────────────────┘
```

### 1.3 상태별 표시

| 상태 | 표시 | 버튼 활성화 |
|------|------|-----------|
| 첫 실행 (데이터 없음) | "스캔을 시작해 주세요" | [스캔 시작] O, [최근 스캔] X |
| 스캔 진행 중 | "스캔 진행 중... (45%)" + 진행바 | 모두 비활성화 |
| 스캔 완료됨 | 요약 통계 + "새 스캔 시작" 버튼 | 모두 활성화 |

## 2. S2: 폴더 선택 (Folder Selection)

### 2.1 목적

스캔 대상 폴더를 선택하고, 스캔 옵션(전체/기간별 등)을 설정 후 스캔 시작

### 2.2 주요 컴포넌트

#### 헤더

```
┌─────────────────────────────────────────────────────┐
│  OptiShot > 폴더 선택            [⚙] [?]          │
└─────────────────────────────────────────────────────┘
```

#### 폴더 선택 영역

```
┌─────────────────────────────────────────┐
│  📁 스캔 대상 폴더                      │
├─────────────────────────────────────────┤
│                                         │
│  ☑ C:\Users\User\Pictures               │
│    └─ 자동 포함: 모든 하위 폴더        │
│                                         │
│  ☑ \\NAS\shared\Photos                 │
│    └─ 자동 포함: 모든 하위 폴더        │
│                                         │
│  ☑ D:\External_HDD\Backup              │
│    └─ 자동 포함: 모든 하위 폴더        │
│                                         │
│  [+ 폴더 추가]  [✖ 제거]  [↑↓ 정렬]   │
│                                         │
└─────────────────────────────────────────┘
```

**폴더 추가 인터랙션**
1. [+ 폴더 추가] 클릭
2. FolderBrowserDialog 띄우기
3. 선택된 경로 목록에 추가

#### 스캔 옵션 영역

```
┌─────────────────────────────────────────┐
│  🔧 스캔 옵션                          │
├─────────────────────────────────────────┤
│                                         │
│  스캔 모드:                             │
│  ◉ 전체 스캔 (모든 파일)               │
│  ◯ 기간별 스캔                         │
│  ◯ 폴더별만 스캔                       │
│  ◯ 증분 스캔 (신규 + 변경분)         │
│                                         │
│  기간 지정 (기간별 선택 시만):        │
│  시작: [2024-01-01] [📅]              │
│  종료: [2024-12-31] [📅]              │
│                                         │
│  [▼ 고급 설정]                        │
│                                         │
└─────────────────────────────────────────┘
```

#### 고급 설정 (토글)

```
┌─────────────────────────────────────────┐
│  ⚙ 고급 설정                            │
├─────────────────────────────────────────┤
│                                         │
│  pHash 임계값: [8] ↑↓                 │
│  (낮을수록 민감, 4~16)                │
│                                         │
│  SSIM 임계값: [0.85] ↑↓               │
│  (0.5~0.95)                            │
│                                         │
│  Time-Window: [1] 시간 ↑↓             │
│  (0~24시간, 촬영 시각 범위)          │
│                                         │
│  병렬 스레드: [8] ↑↓                  │
│  (2~16, CPU 코어 수 권장)            │
│                                         │
│  ☑ 보정본 감지 활성화                 │
│                                         │
└─────────────────────────────────────────┘
```

#### 하단 버튼

```
┌─────────────────────────────────────────┐
│  [◀ 취소]        [스캔 시작 ▶]         │
└─────────────────────────────────────────┘
```

### 2.3 유효성 검사

- 폴더 최소 1개 필수 (선택 없으면 [스캔 시작] 비활성화)
- 경로 존재 여부 확인 (존재하지 않으면 경고)
- 기간 입력 검증 (시작 ≤ 종료)

## 5. S5: 내보내기 (Export)

### 5.1 목적

선택된 베스탈들을 목표 폴더로 Copy/Move

### 5.2 컴포넌트

#### 헤더

```
┌─────────────────────────────────────────────────────┐
│  OptiShot > 내보내기                  [⚙] [?]     │
└─────────────────────────────────────────────────────┘
```

#### 요약

```
┌─────────────────────────────────────────┐
│  📊 내보낼 베스탈                       │
├─────────────────────────────────────────┤
│  파일 수: 5,000개                       │
│  총 용량: 10GB                          │
│  예상 소요 시간: 약 5분                │
│  (네트워크 속도에 따라 다를 수 있음)   │
└─────────────────────────────────────────┘
```

#### 대상 폴더

```
┌─────────────────────────────────────────┐
│  📂 대상 폴더 지정                      │
├─────────────────────────────────────────┤
│                                         │
│  경로: [D:\Curated Photos           ] │

│  [찾아보기 ▼] [경로 확인] [새 폴더] │

│  ☑ 폴더 자동 생성 (없으면)            │
│                                         │
└─────────────────────────────────────────┘
```

#### 액션 선택

```
┌─────────────────────────────────────────┐
│  ⚙ 내보내기 액션                        │
├─────────────────────────────────────────┤
│                                         │
```
