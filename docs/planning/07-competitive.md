# 07-competitive.md: Competitive Analysis

## MVP 캡슐 (요구사항 핵심 10줄)

> **목표**: 흩어진 추억을 안전하게 한 곳에 보존  
> **페르소나**: NAS+HDD+스마트폰에 20만+ 장 분산된 IT 개발자  
> **핵심 기능**: F1 중복감지(2-Stage), F2 베스탈 선별(AI추천+사용자확인), F4 내보내기(단순복사)  
> **North Star**: 중복 그룹 발견율 95%+ / 오탐율 5% 미만  
> **제약**: 20만 장 전체 스캔 30분 이내, 폴더별/기간별 선택 스캔, 증분 스캔  
> **위험**: 보정본 감지 정확도 (pHash 임계값)  
> **완화 전략**: Hamming Distance 임계값 A/B 테스트 (8 vs 10 vs 12)  
> **범위 외**: 메타데이터 통합(v2), 동영상(v2), 모바일(v2)  
> **Input Metric**: 스캔 완료율, 사용자 판정 완료율  
> **다음 단계**: `/screen-spec`으로 화면 명세 생성

## 1. 시장 정의

### 1.1 시장 규모

**TAM (Total Addressable Market)**
- 전 세계 PC 사용자: ~2억 명
- 사진 백업/정리에 관심 있는 사용자: ~5,000만 명
- 중복 감지 도구 사용자: ~500만 명

**SAM (Serviceable Addressable Market)**
- Windows 사용자: ~15억 명
- 사진 > 10,000장 보유자: ~1,000만 명 (Windows)
- NAS 보유자 (한국): ~300만 명

**SOM (Serviceable Obtainable Market)**
- Y1: 5,000명 (오픈소스 + 커뮤니티)
- Y2: 50,000명 (프리미엄 모델)
- Y3: 200,000명 (유료 전환 20%)

### 1.2 경쟁자 분류

| 유형 | 예시 | 특징 |
|------|------|------|
| **클라우드 서비스** | Google Photos, Apple Photos | 클라우드 기반, 블랙박스 AI |
| **데스크톱 전문 도구** | Duplicate Photo Cleaner | 로컬 기반, 기능 제한 |
| **EXIF/메타 관리** | ExifTool, Lightroom | CLI/GUI, 메타데이터 중심 |
| **DIY 솔루션** | Python script, 자체 개발 | 커스텀, 기술 장벽 높음 |

## 2. 주요 경쟁자 분석

### 2.1 Google Photos

**제품 개요**

| 항목 | 내용 |
|------|------|
| 가격 | 무료 (고화질) / 유료 (원본) |
| 플랫폼 | 웹 + 모바일 + 데스크톱 (제한) |
| 스토리지 | 15GB 무료 / $1.99/월 100GB |
| 로컬 처리 | ❌ (클라우드 의존) |
| 중복 감지 | ✅ 자동 (블랙박스) |
| 베스탈 선별 | ✅ 자동만 가능 |

**강점**
- 완성도 높은 AI 추천
- 모든 기기 동기화
- 검색 기능 우수 (얼굴, 장소, 물체)
- 무료 플랜 관대함

**약점**
- 클라우드 종속 (개인정보 우려)
- 로컬 미디어 라이브러리 지원 약함
- NAS 미지원
- 커스터마이징 불가 (블랙박스)
- 사용자 제어권 제한 (자동만)

**OptiShot 차별화**
```
Google Photos vs OptiShot:

Google Photos:
  클라우드 upload → AI 분석 → 추천

OptiShot:
  로컬 파일 → pHash+ORB 분석 → 사용자 최종 판정
  
  핵심: 개인정보 유지 + 사용자 제어
```

**OptiShot 차별화**
```
Duplicate Photo Cleaner vs OptiShot:

Cleaner: 단순 해시 비교 (빠르지만 부정확)
OptiShot: 2-Stage (pHash + ORB/SSIM) + AI (95%+ 정확도)

Cleaner: 무조건 사용자가 선택
OptiShot: AI 추천 + 사용자 확인 (편의성)
```

## 5. Go-to-Market 차별화

### 5.1 OptiShot의 유니크 밸류 프로포지션 (UVP)

```
"내 사진을 내 규칙으로, 단계별로 확인하며 정리하는 로컬 데스크톱 도구"

4가지 기둥:

1️⃣ 로컬 처리 (개인정보)
   - 클라우드 전송 절대 없음
   - 모든 데이터 로컬 저장
   
2️⃣ 정밀 감지 (95%+ 정확도)
   - 2-Stage Detection (pHash + ORB/SSIM)
   - 보정본도 감지
   
3️⃣ 사용자 제어 (신뢰)
   - AI 추천이지만 최종 판정은 사용자
   - 각 단계 확인 가능
   
4️⃣ 멀티 환경 (유연성)
   - Windows + NAS + 외장HDD + 스마트폰 백업
```
